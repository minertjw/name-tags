Place the application icon file here.

Optional header animation:
- header_animation.gif
- This appears at the top-left of the app header.

Expected filename:
- app_icon.ico

Optional fallback filename during development:
- app_icon.png

Windows recommendation:
- Use ICO for the best title bar and taskbar results.
- Include multiple sizes inside the ICO if possible: 16x16, 32x32, 48x48, and 256x256.
- If you start from a PNG, make it square and at least 256x256 before converting to ICO.
